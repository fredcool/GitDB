HOW IT WORKS?

We provide a ready to use schema, database, tables, scripts, etc for designing a project. When the project is being design a git log file will be created which will be having log of every change made in the project from the time the project begins. User can hence monitor the changes made in the project and thereby manage the project.

Further it is here ....

  • Get registered
  • Login to your account
  • Choose the predesigned Project Templates
  • Choose Host link
  • And you are good to go...
  • git init is creates while creating a project
  • Run unit test to check the scripts
  • Check database log and commits by peers
  • Check adding and deleting coloumns and tables
  • Check stored procedures
  • Thereby Manage your Project
  • Get git diff
  • Also receive notification via email
    for all the changes being made in
